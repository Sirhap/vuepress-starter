# 梗
## Java
: 为什么Java程序员总是喜欢写面向对象的代码？

因为他们不想被垃圾回收器清理掉。

## Python
: 为什么Python程序员喜欢喝茶？

因为茶能帮助身体保持缩进。

## JavaScript
: 为什么JavaScript程序员喜欢喝咖啡？

因为他们需要报错时变得更加迅速。

## C++
: 为什么C++程序员总是看起来比较酷？

因为他们无法承受一些其他语言的“垃圾”。

## Go
: 为什么Go程序员总是喜欢说“并发”？

因为他们喜欢在等待服务器响应时同时做很多其他的事情。

## Ruby
: 为什么Ruby程序员总是看起来很有活力？

因为他们可以在每个单词的结尾加上“！”。

## PHP
: 为什么PHP程序员总是被人嘲笑？

因为他们总是在用一些过时的函数和奇怪的标记。
